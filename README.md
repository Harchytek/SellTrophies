# SellTrophies

Configuration library providing balanced economy prices for selling all Valheim trophies via the [TradersExtended](https://thunderstore.io/c/valheim/p/shudnal/TradersExtended/) mod.

<p align="left"><img src="https://imgur.com/XzCrpCT.png" alt="SellTrophies" width="510"></p>

## Features

* **Balanced Economy** : Prices scale naturally with biome difficulty and trophy rarity.
* **58 Trophies Included** : Complete coverage from Meadows to Ashlands.
* **Deep North Ready** : Future-proofed and updated as soon as the next biome drops.

This library was also created to complement [TrophyEater](https://thunderstore.io/c/valheim/p/Harchytek/TrophyEater/), drawing inspiration from the UsefulTrophies mod for players seeking a similar experience.

## Installation
1. Install [BepInExPack Valheim](https://valheim.thunderstore.io/package/denikson/BepInExPack_Valheim/) and [TradersExtended](https://thunderstore.io/c/valheim/p/shudnal/TradersExtended/).
2. Place `shudnal.TradersExtended` folder in your `BepInEx/config` folder.
3. Launch the game and you can sell the trophies to traders.
